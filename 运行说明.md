1. 目前可以直接运行 bash tools/weekly_refresh_and_push.sh
    - 脚本会自动更新每个地点最近7天的鸟单，下载新的鸟，创建不同地点最新的文件夹并把鸟的json放进去，然后push到github上
    - 如果有新鸟，通常需要20分钟左右来下载并上传图片
    - 如果要新增地点，可以先发送以下prmopt
        - 请在birdreport_locations.json中添加地点“植物园”，使周更新时可以自动更新该地点

2. 需要关注terminal中的调试信息
    2.1 是否有下载或上传失败的新鸟，如果有失败的
        - 重新跑一遍 bash tools/weekly_refresh_and_push.sh
        - 如果还失败，就把失败的扔给ai，问为什么，慢慢debug
        - 或者直接自己去下载图片到本地，放在images里，然后重新跑一遍
    2.2 如果有新鸟上传成功
        - 需要去examples的html文件里，筛选一下是否有不合适的图片
        - 在html中选中复制要删除的图片信息，回来复制给ai，并输入以下prompt
            请把以下图片从本地图片、@cloudinary_uploads 、
            @location_birds 、html引用以及其他任何出现的地方都删除
            可使用的工具脚本
            delete_cloudinary_by_list.py - 从 Cloudinary 删除
            cleanup_references_by_list.py - 从 JSON 文件清理引用
            delete_local_images_by_list.py - 删除本地图片文件
            update_gallery_from_cloudinary.py - 重新生成 HTML  
        - 删除完之后重新跑一遍 bash tools/weekly_refresh_and_push.sh
